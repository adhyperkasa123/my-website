## Apa Itu User?

### Definisi User

**User** adalah individu yang akan menggunakan sistem. Setiap user dapat diatur pada level tertentu sesuai fungsi atau perannya, seperti **role** dan **permission** (yang akan dijelaskan pada bagian selanjutnya).

Setiap user memiliki akun tersendiri. Pada bagian ini, Anda akan mendapatkan panduan mengenai tata cara membuat **User** di dalam sistem.

## Mengelola User

Bagian berikut menjelaskan langkah-langkah untuk membuat dan mengelola **User** di sistem.  
Pada contoh ini digunakan akun **Super Admin**, yang memiliki akses penuh untuk mengatur seluruh pengguna.
