## Apa Itu Purchase Request User (PR User)

### Definisi PR User

**Purchase Request User** adalah fitur dalam sistem ERP yang digunakan oleh departemen atau user internal untuk mengajukan permintaan pembelian barang atau jasa yang dibutuhkan dalam operasional perusahaan.

Modul ini merupakan tahap awal proses **Procurement**, di mana user dapat membuat permintaan, mengisi detail kebutuhan seperti item, kuantitas, spesifikasi, serta alasan permintaan. Setelah itu, PR akan dikirim untuk proses review dan persetujuan oleh pihak yang berwenang.

Dengan adanya PR User, proses permintaan pembelian menjadi lebih terstruktur, transparan, dan terdokumentasi dengan baik sehingga dapat mengurangi kesalahan dan mempercepat proses pengadaan.
