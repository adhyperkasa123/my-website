# Setting Stock Entry

Dokumen ini menjelaskan cara membuka menu **Stock Entry**, memahami tampilan halaman utama, serta menggunakan fitur **Export** dan **Add Stock Entry**.

---

# 1. Masuk ke Menu Stock Entry

Setelah user login dan memilih Workspace, lakukan langkah berikut untuk membuka menu **Stock Entry**:

![Masuk ke Workspace Stock Entry](image.png)

### Langkah-langkah membuka menu Stock Entry
1. Buka **sidebar kiri**.
2. Klik menu **Procurement**.
3. Pilih submenu **Stock Entry**.
4. Sistem akan menampilkan halaman utama Stock Entry.

Menu ini digunakan untuk melakukan proses pencatatan perpindahan stok seperti *Material Issue*, *Material Receipt*, *Stock Transfer*, dan *Stock Adjustment*.

![Shortcut Stock Entry](image-1.png)

---

# 2. Tampilan Halaman Stock Entry

Halaman **Stock Entry** menampilkan semua transaksi pergerakan stok yang telah dicatat dalam sistem.  
Pada halaman ini, Anda dapat:

- Mencari transaksi menggunakan **Search Transaction number**.  
- Menambahkan transaksi baru melalui tombol **+ Add Stock Entry** di kanan atas.

## Informasi yang Ditampilkan
Setiap baris pada daftar Stock Entry menampilkan informasi berikut:

1. **Transaction Number**  
2. **Transaction Date**   
3. **Status** 
4. **Type** 
4. **Action**

---

![Tampilan Halaman Stock Entry](image-2.png)

---

# 3. Masuk ke Halaman Create Stock Entry

Untuk membuat transaksi Stock Entry baru, klik tombol **+ Create Stock Entry**.  
Sistem akan menampilkan form untuk membuat Stock Entry sesuai kebutuhan.

![Masuk ke  Create Stock Entry](image-3.png)

> ⚠️ **Catatan:**  
> Kolom dengan tanda **`*` merah** adalah **wajib diisi**.  
> Kolom tanpa tanda bintang bersifat **opsional**.
