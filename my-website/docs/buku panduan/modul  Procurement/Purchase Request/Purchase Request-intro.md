##  Apa Itu Purchase Request

### Definisi Purchase Request

Modul **Purchase Request** di ADFood adalah tahap lanjutan dari **PR User** yang dikelola oleh tim procurement untuk meninjau, memvalidasi, dan mengonsolidasikan permintaan pembelian dari user internal. Setelah item dari PR User ditinjau dan data pendukung dilengkapi, **Purchase Request** dipublikasikan (**Published**). Pada saat itulah, sistem otomatis mengubah status PR User terkait menjadi **“To PO”**, yang menandakan permintaan siap dilanjutkan ke tahap pembuatan **Purchase Order (PO)**

### Cara Melakukan Transaksi Purchase Request

Berikut ini akan dijelaskan langkah demi langkah cara melakukan transaksi **Purchase Request** di sistem ERP ADFood. Pada contoh ini, Workspace yang digunakan adalah **Company Branch** dengan nama *Supporting Office Kebagusan Raya*. Pastikan Anda sudah login ke sistem ERP ADFood, kemudian pilih **Workspace** yang sesuai.
