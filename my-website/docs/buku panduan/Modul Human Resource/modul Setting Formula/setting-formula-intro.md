# Setting Formula Payroll 

##  Apa Itu Setting Formula

###  Pengertian Setting Formula
**Setting Formula** pada ERP ADFood adalah fitur yang digunakan untuk mengatur rumus perhitungan gaji karyawan secara otomatis. Dengan fitur ini, perusahaan dapat menentukan cara perhitungan komponen-komponen seperti:

- Basic Salary (Gaji Pokok)
- Allowance (Tunjangan)
- Overtime Pay (Lembur)
- Bonus
- Deductions (Potongan seperti pajak, BPJS, pinjaman)

Setting Formula berguna untuk memastikan perhitungan payroll:

✅ Konsisten  
✅ Akurat  
✅ Otomatis  
✅ Sesuai kebijakan perusahaan

###  Cara Mengelola Setting Formula
User dapat mengelola formula melalui:

**Human Resource → Payroll → Setting Formula**

Di dalam menu ini, user dapat membuat, menyalin, mengedit, serta menghapus formula payroll sesuai kebutuhan.
